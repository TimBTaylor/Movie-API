The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Description URL Method Request Response
Return a list of all movies '/movies' GET None A JSON object with all the information of the movies
Return data about a single movie '/movies/:title' GET None A JSON object with all information about the single movie
Return a description about the genre of the movie '/movies/:title/genre' GET None An HTML response with a description of the genre
Return data about a director(bio, birth year, death year) '/movies/director/:name' GET None An HTML response with a description of the director
Allow new users to register '/users' POST {
"username": "[username]"
, "password": "[password]"
, "email": "[email]"
, "birthday": "[birthday]"
} 		An HTML response that the account has been created
Allowing users to update their username '/users/:username/:newusername' PUT None A JSON response that contains the users information
Allow users to add a movie to their list of favorites '/users/:username/movies/:movieID' POST { "movie":"movieName"
, "genre":"movieGenre"
, "director":"movieDirector" } 		A text showing that the movie has been added
Allow users to remove a movie from their favorites '/users/:username/movies/:movieID' DELETE { "movie":"movieID" } A text showing that the movie has been removed
Allowing users to deregister '/users/:username' DELETE Username A text showing that the user has been removed
