The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Description URL Method Request Response
Return a list of all movies '/movies' GET None A JSON object with all the information of the movies
Return data about a single movie '/movies/:title' GET Title A JSON object with all information about the single movie
Return a description about the genre '/movies/genres/:Genre' GET None A response with a description of the genre
Return data about a director(bio, birth year, death year) '/movies/directors/:Director' GET Directors name A response with a description of the director
Allow new users to register '/users' POST {
"username": "[username]"
, "password": "[password]"
, "email": "[email]"
, "birthday": "[birthday]"
} 		A response of the information entered indicating the account has been created
Allowing users to update their information '/users/:Username' PUT {
"username": "[username]"
, "password": "[password]"
, "email": "[email]"
, "birthday": "[birthday]"
} 		{
"username": "[updatedUsername]"
, "password": "[updatedPassword]"
, "email": "[updatedEmail]"
, "birthday": "[updatedBirthday]"
}
Allow users to add a movie to their list of favorites '/users/:Username/Movies/:MovieID' POST Username, movieID A text showing that the movie has been added
Allow users to remove a movie from their favorites '/users/:username/movies/:movieID' DELETE Username, movieID A text showing that the movie has been removed
Allowing users to deregister '/users/:Username/Movies/:MovieID' DELETE Username A text showing that the user has been removed or not found
Allowing admin to return all information about a user '/users/:Username' GET Username A text displaying all the information about that user
Allows admin to return all information about all users '/users' GET None A text showing all information about each user
